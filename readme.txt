=== WooCommerce Valutec Giftcards ===
Author: Caleb Hearon
Tags: woocommerce
Requires at least: 4.4
Tested up to: 4.5
Requires WooCommerce at least: 2.4.12
Tested WooCommerce up to: 2.5.3

Allow people to pay with Valutec gift cards at the checkout screen

Full documentation: See README.md

== Installation ==

1. Upload the entire 'woocommerce-valutec' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
